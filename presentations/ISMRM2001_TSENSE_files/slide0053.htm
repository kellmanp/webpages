Sensitivity Map Estimates
(surface coil estimates)
•In order to estimate sensitivity maps using surface coils only, it necessary to remove the image magnitude and phase components.
•This is necessary for 2 reasons:
1) Any dynamic component in the image will cause the desired image to be temporal modulated, and even filtered.
2) It is often desirable to apply some degree of spatial smoothing.
•I use the following scheme:
•The sample correlation matrix is computed at each pixel which removes the image phase, but preserves the coil phase differences.
•At this point spatial smoothing can be applied
•The sensitivities are then computed as the dominant eigenvector.
•And finally used to compute SENSE coeffs incorporating noise statistics.
•The phase map illustrates how much variation in phase there is in the heart over a few cardiac cycles. The final estimate of relative coil phase has the temporal modulation removed.
