•This slide descibes the phase sensitive reconstruction
•The magnitude images for the T1-weighted IR image and the reference are shown.
•The phase sensitive image is derived by rotating the phase of each pixel to remove the background phase, and then simply using the real image.
•Another benefit of this method is a slight improvement in SNR in regions of low SNR.
•We also use the multi-coil reference images to estimate complex B1-maps for optimum phase array combining.
•Both the IR and reference images are combined optimally
•before estimating the background phase, and
•before phase sensitive reconstruction.
