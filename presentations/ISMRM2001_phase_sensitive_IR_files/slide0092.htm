•This slide descibes the normalization of surface coil intensity, which uses the magnitude of the reference image.
•The phase sensitive image is normalized by dividing pixel-by-pixel using the reference magnitude.
•A small amount of spatial smoothing is used to improve the SNR of the reference.
•This allows the level to be adjusted to set the null point.
•Using a 5 degree flip angle for the reference image, the reference variation due to T1 weighting is on the order of 10 to 15% variation, depending on the heart rate.
