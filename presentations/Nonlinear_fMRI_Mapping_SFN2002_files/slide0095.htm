•A simulation was performed using a cascade model of 2 nonlinear systems as described, each a Wiener system composed of a linear dynamic system followed by a memoryless nonlinearity (i.e., L1N1L2N2).
•The 1st system (L1) was bandpass and had a fast response, and the 2nd system (L2) was a lowpass with a slow response on the order of several seconds. Full-wave rectification non-linearities were used.
•On the left is the simulation for bullseye reversals. The intensity is 1,0 corresponds to a check that is initially white. The ±1 signed contrast corresponds to the output of the spatial Laplacian resulting from retinal processing. The linear filter L1 causes spikes at the transitions and a small proportional or DC response is produced as well.
•Various waveforms are displayed for the 1st 16 m-seqence bits, and the full period cross-correlation is shown on the bottom.
•After rectification the proportional response is constant & no longer correlates with the m-sequence. The second order response corresponding to the transitions is evident.
•Next, consider the stimulus paradigm on the right. A 200 msec gap was introduced between stimuli as a decorrelating delay to discriminate between fast and slow systems. This gap is much > the fast neuronal response but much < than the several sec hemodynamic response. A uniform gray was presented during the gap.
•In this case, there are periodic spikes which do not result in a second order correlation.
•If the system had been a single cascade L2N2 with slow L2, the 2nd order response would not have disappeared.
