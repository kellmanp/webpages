This slide illustrates real time imaging with and without acceleration. In real-time imaging, the complete set of phase encode for each image is acquired in a free-running manner which does require ECG triggering or breath-holding.  For fully sampled K-space (shown at the top)  the spatial and/or temporal resolution may not be adequate, thus accelerated imaging may be used by under-sampling the phase encodes to go faster as shown in lower diagram.  The boxes represents the time of each image.   Under-sampling causes wrap artifacts which may be unwrapped using the SENSE reconstruction method. This example used 4-to-1 acceleration to go 4x faster.
