for phase encode order
 Benefits include reduced distortion and artifacts due to:
• off-resonance, flow, T2* decay, …
• reduced EPI blip size (as compared to interleaved)
 Eliminates need for echo shifting
• reduces TR
• additional cardiac phases (breath-held cine)
Conclusions
•In conclusion, the adaptive phased array ghost cancellation method permits new EPI design strategies which have benefits in reduced artifacts increased acquisition effiency.
•Additionally, with greater design flexibility, you can use more optimal time ordering of shots, as for example in contrast enhanced imaging using saturation recovery
