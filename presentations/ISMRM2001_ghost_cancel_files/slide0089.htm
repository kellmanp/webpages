Adaptive Calculation of
Array Combiner Complex Coefficients
•The phased array ghost cancellation method is predicated on knowing the complex coil sensitivity profiles which must be estimated.
•Let me describe an adaptive method that we use to estimate the coil sensitivities which is applicable to cine or real time acquisition of multiple image frames.
•A time varying phase encode order is used, for which the echo order is periodically varied in a cyclic fashion.
•For instance, in the example shown with ETL=3, the order of echoes within each shot is 1,2,3 then 2,3,1 then 3,1,2 and so on.
•After 3 frames, each ky line has been acquired at all echo delays. There are a number of ways to then reconstruct a lower temporal resolution, ghost suppressed image which can be used for calculating the phased array coefficients. In this illustration we simply use a low pass temporal filter.
•The coefficients are then applied to the various images and their shifts to separate the ghosts, followed by combining.
